# casestudy-webtech
CaseStudy for IBZ-Web-Technology

# Setup

Install nodejs: https://nodejs.org/dist/v5.3.0/node-v5.3.0-x64.msi

```
npm install npm -g
npm install node-gyp -g
node-gyp rebuild
npm install
```

# Run
```
npm run lite
```
